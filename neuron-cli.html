neuron-cli

A terminal knowledge manager that doesn't get in your way.

Ver en GitHub Instalación rápida
# Homebrew (macOS / Linux)
brew install steevin/tap/neuron
# Go install
go install github.com/steevin/neuron-cli@latest
# Build from source
git clone https://github.com/steevin/neuron-cli
cd neuron-cli && make build

Acerca del proyecto

~ neuron 
❯ neuron                         # abre la TUI
❯ neuron init                    # asistente de configuración
❯ neuron add "standup notes" --tag work
✓ Nota creada: standup notes.md
❯ neuron search kubernetes     # búsqueda fuzzy en tu vault
❯ neuron daily                   # abre/crea la nota del día
❯ neuron sync                    # commit + push con Git

neuron es un gestor de notas local-first, compatible con Obsidian, diseñado para vivir en tu terminal. Mantiene tu vault de Markdown exactamente donde está — sin migraciones, sin nube, sin suscripciones. Solo acceso rápido, guiado por teclado, a tus notas desde cualquier shell.

Características

TUI Interactiva

Navega tu vault completo con una Terminal User Interface fluida. Presiona / para abrir la paleta de comandos fuzzy, o ? para ver todos los atajos de teclado.

Prompts Inteligentes

¿Olvidaste pasar un argumento? neuron te lo pide de forma interactiva con una UI hermosa gracias a huh. Nunca más mires un error de ayuda genérico.

Sincronización Git

Un solo comando neuron sync hace commit y push de tu vault. Historial de versiones completo sin salir de la terminal.

Servidor MCP

Expone tu vault como un Model Context Protocol server, permitiendo que agentes de IA como Claude o Gemini lean y escriban en tus notas de forma segura.

Notas Diarias

neuron daily crea o abre la nota del día al instante, con la plantilla que definas. Perfecto para journaling o standups.

Output Colorizado

Salidas limpias y coloridas con lipgloss y spinners animados con bubbles. La terminal, pero con estilo.

Comandos

Comando
Descripción
neuron
Abre la TUI principal (modo por defecto)
neuron init
Asistente interactivo de configuración inicial
neuron add [título]
Crea una nota nueva; te pide el título si falta
neuron edit [nota]
Abre la nota en $EDITOR
neuron search [query]
Búsqueda fuzzy en todo el vault
neuron daily
Crea o abre la nota diaria de hoy
neuron sync
Commit y push del vault a Git
neuron mcp
Inicia el servidor MCP para agentes de IA
