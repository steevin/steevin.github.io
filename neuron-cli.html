Inicio Rápido

1

Instala Neuron

Consigue Neuron a través de Homebrew, o mediante descarga directa en segundos.

2

Inicializa tu Bóveda

Apunta Neuron a un directorio existente de Obsidian, o crea una bóveda desde cero.

3

Inicia la TUI

Navega, crea y organiza notas. Presiona ? dentro de la TUI para ver los atajos.

~ neuron 
❯ neuron                         # abre la TUI
❯ neuron init                    # asistente de configuración
❯ neuron add "standup notes" --tag work
✓ Nota creada: standup notes.md
❯ neuron list -q "kubernetes"  # búsqueda fuzzy en tu vault
❯ neuron today                   # abre/crea la nota del día
❯ neuron sync                    # commit + push con Git

Filosofía del Proyecto

El teclado manda

Tus manos nunca deberían dejar la fila inicial. Cada acción —desde buscar notas hasta mover carpetas y extraer bloques de código— está a un atajo de distancia.

Privacidad por defecto

Tus pensamientos son tuyos. Neuron funciona 100% offline. No realiza tracking, no sube tus notas a la nube y no requiere crear una cuenta.

Libertad de formato

Creemos en Markdown plano con frontmatter YAML estándar. Si decides dejar de usar Neuron mañana, tus notas siguen siendo archivos de texto legibles.

Cero bloqueo de datos

Trabaja directamente con tu directorio local. Puedes abrir tus notas en Obsidian, VS Code o Vim en cualquier momento sin importación o exportación previa.

Velocidad sin fricciones

Abre, busca, crea y organiza notas en milisegundos usando una aplicación nativa escrita en Go y optimizada para el máximo rendimiento.

Listo para IA

Realiza búsquedas semánticas conceptuales integrando Ollama, o expón toda tu base de conocimientos a agentes inteligentes mediante el protocolo MCP.

Características principales

TUI Interactiva

Navega tu vault completo con una Terminal User Interface fluida. Presiona / para la paleta fuzzy o ? para ver los atajos.

Prompts Inteligentes

neuron te pide cualquier argumento faltante de forma interactiva con huh. Sin errores de ayuda genéricos.

Sincronización Git

neuron sync hace commit y push de tu vault en un solo comando.

Servidor MCP

Expone tu vault como un Model Context Protocol server para agentes de IA como Claude o Gemini.

Notas Diarias

neuron daily crea o abre la nota del día al instante con tu plantilla preferida.

Output Colorizado

Salidas limpias y coloridas con lipgloss y spinners animados con bubbles.

Metodología PARA

Entiende la estructura Projects, Areas, Resources, Archive. Incluye un selector interactivo de carpetas.

Búsqueda Dual

Búsqueda de texto completo (BM25) incorporada, y búsqueda semántica mediante embeddings con Ollama.

Flujo de Pegapapeles

ctrl+v sobre una nota pega el contenido directamente. Soporta bracketed paste para títulos y cuerpo.

Wikilinks y Grafo

Soporte completo para [[wikilinks]] y #tags. Presiona g para ver un resumen de tu grafo de conocimiento.

Sistema de Plantillas

Crea plantillas para tus notas y úsalas bajo demanda con --template, o automáticamente para notas diarias.

Temas y Splash Screen

Pantalla de inicio con estadísticas. Cambia en vivo entre modo oscuro (Tokyo Night) y claro (GitHub).

# Homebrew (macOS / Linux)
brew install steevin/tap/neuron
# Instalar binario precompilado mediante Curl
curl -sSfL https://github.com/steevin/neuron-cli/releases/latest/download/neuron_$(uname -s)_$(uname -m).tar.gz | tar -xz -C /usr/local/bin neuron
# Instalar usando Go (requiere entorno de desarrollo de Go)
go install github.com/steevin/neuron-cli@latest
# Compilar directamente desde el código fuente
git clone https://github.com/steevin/neuron-cli
cd neuron-cli && make build
Para actualizar, ejecuta: brew upgrade steevin/tap/neuron o vuelve a descargar mediante Curl.

Comandos y Atajos

Comando
Descripción
neuron
Abre la TUI interactiva (comportamiento por defecto).
neuron init
Inicia el asistente interactivo para configurar tu bóveda.
neuron add
Solicita un título de nota y activa el selector de carpetas PARA.
neuron add "standup" --tag work
Crea una nota con una etiqueta inicial y abre el selector.
neuron add "1. Projects/API"
Crea la nota en una ruta explícita saltándose el selector interactivo.
neuron add "Config" --file nginx.conf --code
Crea una nota directamente importando un archivo externo como código.
cat s.py | neuron add "Script" --code python
Crea una nota importando código desde una tubería o pipeline (pipe).
neuron edit "standup notes"
Abre la nota seleccionada directamente en tu $EDITOR predeterminado.
neuron today
Abre o crea automáticamente la nota diaria de hoy.
neuron list -q "kubernetes"
Realiza búsquedas fuzzy clásicas o búsquedas conceptuales (IA).
neuron move "standup" projects
Mueve la nota directamente a tu carpeta de Proyectos.
neuron attach "notes" ./img.png
Copia y adjunta un archivo local a la carpeta de assets y añade el link.
neuron links "notes"
Busca, extrae y abre los enlaces de la nota en el navegador.
neuron sync --pull
Sincroniza tu bóveda realizando un Git pull y Git push en un paso.
neuron stats
Muestra estadísticas globales de la bóveda (total de notas, etiquetas, enlaces).
neuron config set editor nvim
Establece tu editor de terminal predeterminado de manera permanente.
neuron config set theme dark
Configura el tema permanente para la interfaz visual de la TUI.
neuron mcp
Inicia el servidor local de Model Context Protocol.
Comando TUI
Descripción

Crea una nueva nota (con selector de carpetas).
/today

/copy, /c
Copia la nota actual al portapapeles.

Teclas Generales
Acción
j / k o ↑ / ↓
Mueve la selección hacia arriba o hacia abajo en la lista.
Enter
Selecciona la nota activa o confirma una acción.
Tab / Shift+Tab
Alterna el foco entre el panel de navegación y la vista previa de la nota.
n
Crea una nota nueva abriendo el selector de carpetas PARA.
e
Abre el archivo seleccionado en tu editor de código preferido.
ctrl+v
Pega instantáneamente el contenido de tu portapapeles al final del archivo.
c / y
Copia o extrae los bloques de código contenidos en la nota actual.
/
Abre la paleta de comandos interactivos.
s
Sincroniza los cambios con tu repositorio Git.
ctrl+g
Muestra estadísticas del grafo de conexiones del conocimiento.
?
Despliega un overlay de ayuda con todos los atajos de teclado.
q
Termina la ejecución de la TUI.
Durante Selección de Carpeta
Acción

Esc

Durante Modo Copiar Código
Acción
← → / h l / ↑ ↓ / j k
Navega entre los diferentes bloques de código contenidos en la nota.

Integración con Agentes de IA (MCP)

Neuron expone tu bóveda de notas locales a través del Model Context Protocol (MCP). Puedes configurarlo en clientes compatibles (Claude Desktop, Cursor, Antigravity) para permitir a las inteligencias artificiales leer, crear, resumir u organizar tus notas en tiempo real sin salir de tu ventana de chat. 

{
  "mcpServers": {
    "neuron": { "command": "neuron", "args": ["mcp"] }
  }
}
