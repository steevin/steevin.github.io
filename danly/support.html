Soporte

Última actualización: 22 de junio de 2026

Estamos aquí para ayudarte con Danly, una app de diario personal para guardar, organizar y recordar momentos importantes.

Para soporte, preguntas o solicitudes relacionadas con tu cuenta, escríbenos a danly@steevin.com.

Contacto

Envía tu consulta a danly@steevin.com. Incluye una descripción breve del problema, el dispositivo que usas y, si aplica, los pasos para reproducirlo.

Responderemos tan pronto como sea posible.

Ayuda frecuente

  • Si no puedes iniciar sesión, revisa que estés usando el mismo método de acceso con el que creaste tu cuenta.
  • Si activaste sincronización, asegúrate de tener conexión a internet y una sesión activa.
  • Si usas recuerdos cifrados en otro dispositivo, necesitarás tu frase de recuperación.
  • Si una compra premium no aparece, abre Ajustes dentro de Danly y usa la opción para restaurar compras.

Privacidad y datos

Puedes revisar cómo Danly maneja tus datos en la política de privacidad.

Para solicitudes de acceso, corrección, exportación o eliminación de datos, escríbenos a danly@steevin.com.

Reporte de problemas

Cuando reportes un error, ayuda incluir la versión de iOS, el modelo del dispositivo, la versión de Danly y qué estabas intentando hacer antes de que ocurriera el problema.

No envíes información sensible ni contenido privado de tu diario salvo que sea necesario para resolver tu solicitud.
