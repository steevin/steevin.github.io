Política de privacidad

Última actualización: 22 de junio de 2026

Danly es una app de diario personal. Puedes usarla en modo local y, si decides iniciar sesión, sincronizar tus recuerdos cifrados entre dispositivos.

No vendemos tus datos personales, no mostramos publicidad y no usamos tus recuerdos para seguimiento publicitario.

Datos que puedes guardar

Danly puede guardar notas, títulos, fechas, etiquetas, personas escritas por ti, estados de ánimo, pendientes, fotos, audios, transcripciones, favoritos y diarios consolidados.

Como los campos de texto y audio son libres, puedes escribir o grabar información personal o sensible. Tú decides qué contenido guardar.

Cuenta y sincronización

Si usas Danly en modo local, tus datos permanecen en este dispositivo salvo las acciones que tú inicies, como exportar o compartir.

Si creas una cuenta o inicias sesión, Danly puede guardar tu identificador de usuario, correo electrónico, nombre visible, rol de acceso y estado premium para autenticarte, habilitar sincronización y administrar funciones de la app.

Si conectas una cuenta, Danly usa Firebase Authentication para iniciar sesión y Cloud Firestore para sincronizar registros cifrados. Los archivos de audio o imagen se suben cifrados a CloudKit cuando corresponde.

La clave de cifrado vive en tu dispositivo. La frase de recuperación es necesaria para abrir tus recuerdos cifrados en otro dispositivo.

IA y Gemini

Las funciones de IA son opcionales. Si autorizas el procesamiento con IA, Danly puede enviar texto de tus notas o transcripciones a Gemini para detectar ánimo o generar diarios del día.

Danly no envía fotos ni audios para generar el diario del día; usa texto de las notas y transcripciones cuando la función lo requiere.

Danly puede registrar datos técnicos de uso de IA, como la función usada, el modelo, conteos de tokens, fecha, identificador de usuario y correo electrónico, para operar, medir costos y administrar el servicio. Ese registro no guarda el contenido de tus recuerdos.

Puedes desactivar el procesamiento con IA desde Ajustes. Si está desactivado, Danly no enviará tus notas a Gemini para estas funciones.

Compras y premium

Las compras se procesan mediante StoreKit y Apple. Danly no recibe ni almacena el número de tarjeta ni los datos completos de pago.

Danly puede guardar localmente si tu Apple ID tiene una suscripción premium activa para desbloquear funciones de la app. Si tu cuenta tiene un rol premium o administrador, Danly puede guardar ese estado en Firebase y localmente.

Permisos del dispositivo

Danly puede solicitar permisos de micrófono, reconocimiento de voz, fotos, notificaciones y Face ID. iOS te permite conceder, limitar o revocar esos permisos desde Ajustes del sistema.

El reconocimiento de voz usa los servicios de Apple para transcribir tus notas habladas cuando solicitas una transcripción.

Face ID se usa para bloquear el acceso local a la app. Danly no recibe ni almacena datos biométricos.

Exportaciones y borrado

Puedes exportar tus datos desde Ajustes. El archivo exportado queda bajo tu control y puede contener información personal.

También puedes borrar los datos guardados desde Ajustes. Si usas sincronización, Danly intentará eliminar también los registros remotos asociados a tu cuenta.

Proveedores

Danly usa servicios de Apple para StoreKit, CloudKit, reconocimiento de voz y funciones del sistema; Firebase/Google para autenticación, sincronización y registros técnicos; y Gemini para funciones opcionales de IA.

Estos proveedores pueden procesar datos técnicos necesarios para operar sus servicios, como identificadores de app, estado de compra, registros operativos o información de seguridad.

Retención

Conservamos los datos mientras permanezcan en la app o en tu cuenta sincronizada. Puedes borrarlos desde Danly o escribirnos para solicitar ayuda.

Los datos exportados o compartidos fuera de Danly no se eliminan automáticamente cuando borras datos dentro de la app.

Menores

Danly no está dirigida a menores de 13 años. Si crees que un menor nos proporcionó datos personales sin autorización, contáctanos para revisar y eliminar la información correspondiente.

Tus opciones

Puedes usar modo local, desactivar IA, desactivar recordatorios, revocar permisos en iOS, exportar datos, borrar datos y cerrar sesión.

Para solicitudes de acceso, corrección, exportación o eliminación, escríbenos a danly@steevin.com.

Contacto

Si tienes preguntas sobre esta política o sobre tus datos, contáctanos en danly@steevin.com.

Esta política puede actualizarse cuando cambien las funciones, proveedores o requisitos legales de Danly.
